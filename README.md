# Docker-Wordpress

This file will setup Wordpress, MySQL & PHPMyAdmin with a single command. Add the code below to a file called "docker-compose.yaml" and run the command.

## Installation

```
docker-compose up -d
```

## To Tear Down

```
docker-compose down --volumes
```
